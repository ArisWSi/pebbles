0x81a9体面的人不扔垃圾

2026-06-09

我站在十字路口等红灯，旁边一个穿西装的商务男士伸手去包里掏东西，一盒薄荷糖掉了出来。大约三十颗白色小糖丸像弹珠一样滚满了马路。

「其实乱扔垃圾是好事，不然那些靠捡垃圾为生的人就要失业了。」

那句话让我的朋友爆笑——他站到了我这边，开始用这话逗他妈妈。我完全翻转了局势——朋友不再觉得我做了蠢事，而是觉得我做了聪明事。那是我第一次意识到，为一些不真实的事情辩护，既可以很好玩，也可以让人印象深刻。

在这个八十亿人的星球上生活，最疯狂的事情之一就是：如果你有一个麻烦，在某些情况下你基本可以把这个麻烦像挥棒球棍一样甩进人群，让它砸在某个倒霉路人的头上。对你来说，麻烦等于消失了。对那个被砸中的人来说——麻烦像是凭空出现的。多有趣啊！

// Perlin noise 实现的 ASCII 字符动画
fn animHash(x: f32) -> f32 {
    ((sin(x * 91.3458) * 47453.5453) % 1.0 + 1.0) / 2.0
}

作为机器学习从业者，我喜欢把内心的那个小声音想象成local gradient signal。它工作的层面差不多就是：「我不喜欢街上满是垃圾」「垃圾只有人丢了才存在」「我是人」「所以我不应该丢垃圾」。
